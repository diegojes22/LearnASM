# Aprendiendo ensamblador desde 0

Hola, en este repositorio estara todo lo que vaya haciendo en turbo ensamblador. Este repositorio puede ser de utilidad para aquellos que cursan la materia de **Lenguajes de interfa** en el **TecNM**, a si que espero que esto sea de ayuda.

### Entorno
El curso que tomo de ensamblador utiliza **Turbo ensamblador** para **x86**, y para ejecuta el codigo sin restricciones y de forma segura utilizaremos **dosbox**. 

Para la instalacion de dosbox puedes ir al [Sitio web oficial](https://www.dosbox.com/download.php?main=1) y ahi descargas el instalador, o en este mismo repositorio puedes encontrar el instalador para sistemas Windows. Con respecto a al ensambaldor y al entazador de **TASM**, estos se encuentran en la carpeta de `/resources`.
